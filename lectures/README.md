# Лекційні матеріали

Ця тека призначена для переносу матеріалів з `docx/.NET - OOP.docx` у Markdown.

Правило переносу:

- копіювати зміст книги максимально 1 в 1;
- не переписувати матеріал власним стилем без потреби;
- не скорочувати пояснення;
- виправляти лише очевидні помилки, пошкоджений перенос з DOCX, зламані приклади коду та явні технічні неточності;
- код оформлювати Markdown-блоками;
- усі редакторські втручання фіксувати в журналі переносу.

## Структура для сайту

Для відображення на сайті лекції додатково розрізані на окремі підрозділи в `lectures/sections/`.

Формат: `RR-PP-slug.md`, де `RR` - номер розділу, `PP` - номер підрозділу.

Наприклад:

- `02-04-typy-danykh.md` -> `2.4. Типи даних`
- `02-10-yavni-ta-neyavni-peretvorennia.md` -> `2.10. Явні та неявні перетворення`
- `03-01-klasy-ta-obiekty.md` -> `3.1. Класи та об'єкти`
