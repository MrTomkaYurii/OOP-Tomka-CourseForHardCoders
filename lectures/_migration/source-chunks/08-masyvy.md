# 2.14. Масиви

Масив представляє набір однотипних даних. Оголошення масиву схоже на оголошення змінної за винятком, що після вказівки типу ставляться квадратні дужки:

```text
тип_змінної[] назва_масиву;
```

Наприклад, визначимо масив цілих чисел:

```csharp
int[] numbers;
```

Після визначення змінної масиву ми можемо надати їй певне значення:

```csharp
int[] nums = new int[4];
```

Тут спочатку ми оголосили масив `nums`, який зберігатиме дані типу `int`. Далі використовуючи операцію `new`, ми виділили пам'ять для 4 елементів масиву: `new int[4]`. Число 4 ще називається довжиною масиву. При такому визначенні всі елементи набувають значення за умовчанням, яке передбачено для їх типу. Для типу `int` значення за промовчанням - 0.

Також ми одразу можемо вказати значення для цих елементів:

```csharp
int[] nums2 = new int[4] { 1, 2, 3, 5 };

int[] nums3 = new int[] { 1, 2, 3, 5 };

int[] nums4 = new[] { 1, 2, 3, 5 };

int[] nums5 = { 1, 2, 3, 5 };
```

Усі перелічені вище методи будуть рівноцінні.

Подібним чином можна визначати масиви та інших типів, наприклад, масив значень типу `string`:

```csharp
string[] people = { "Tom", "Sam", "Bob" };
```

## Індекси та одержання елементів масиву

Для звернення до елементів масиву використовуються індекси. Індекс представляє номер елемента в масиві, при цьому нумерація починається з нуля, тому індекс першого елемента дорівнюватиме 0, індекс четвертого елемента - 3.

Використовуючи індекси, ми можемо отримати елементи масиву:

```csharp
int[] numbers = { 1, 2, 3, 5 };

// Отримання елемента масиву
Console.WriteLine(numbers[3]); // 5

// отримання елемента масиву в змінну
var n = numbers[1]; // 2
Console.WriteLine(n); // 2
```

Також ми можемо змінити елемент масиву за індексом:

```csharp
int[] numbers = { 1, 2, 3, 5 };

// Змінимо другий елемент масиву
numbers[1] = 505;

Console.WriteLine(numbers[1]); // 505
```

І оскільки у нас масив визначений лише для 4 елементів, то ми не можемо звернутися, наприклад, до шостого елемента. Якщо так спробуємо зробити, ми отримаємо помилку під час виконання:

```csharp
int[] numbers = { 1, 2, 3, 5 };

Console.WriteLine(numbers[6]); //! Помилка - у масиві лише 4 елементи
```

## Властивість Length та довжина масиву

Кожен масив має властивість `Length`, яка зберігає довжину масиву. Наприклад, отримаємо довжину вище за створений масив `numbers`:

```csharp
int[] numbers = { 1, 2, 3, 5 };

Console.WriteLine(numbers.Length); // 4
```

Для отримання довжини масиву після назви масиву через точку вказується властивість `Length`: `numbers.Length`.

## Отримання елементів з кінця масиву

Завдяки наявності властивості `Length`, ми можемо обчислити індекс останнього елемента масиву - це довжина масиву - 1. Наприклад, якщо довжина масиву - 4 (тобто масив має 4 елементи), то індекс останнього елемента дорівнюватиме 3. І, використовуючи властивість `Length`, ми можемо легко отримати елементи з кінця масиву:

```csharp
int[] numbers = { 1, 2, 3, 5 };

Console.WriteLine(numbers[numbers.Length - 1]);
// 5 - перший з кінця або останній елемент
Console.WriteLine(numbers[numbers.Length - 2]);
// 3 - другий з кінця чи передостанній елемент
Console.WriteLine(numbers[numbers.Length - 3]);
// 2 - третій елемент з кінця
```

Однак при подібному підході вирази типу `numbers.Length - 1`, сенс яких полягає в тому, щоб отримати якийсь певний елемент з кінця масиву, код обтяжують. І, починаючи, з версії C# 8.0 до мови було додано спеціальний оператор `^`, з допомогою якого можна задати індекс щодо кінця колекції.

Перепишемо попередній приклад, застосовуючи оператор `^`:

```csharp
int[] numbers = { 1, 2, 3, 5 };

Console.WriteLine(numbers[^1]);
// 5 - перший з кінця або останній елемент
Console.WriteLine(numbers[^2]);
// 3 - другий з кінця чи передостанній елемент
Console.WriteLine(numbers[^3]);
// 2 - третій елемент з кінця
```

## Перебір масивів

Для перебору масивів можна використовувати різні типи циклів. Наприклад, цикл `foreach`:

```csharp
int[] numbers = { 1, 2, 3, 4, 5 };
foreach (int i in numbers)
{
    Console.WriteLine(i);
}
```

Тут як контейнер виступає масив даних типу `int`. Тому ми оголошуємо змінну з типом `int`.

Подібні дії ми можемо зробити і за допомогою циклу `for`:

```csharp
int[] numbers = { 1, 2, 3, 4, 5 };
for (int i = 0; i < numbers.Length; i++)
{
    Console.WriteLine(numbers[i]);
}
```

У той самий час цикл `for` більш гнучкий порівняно з `foreach`. Якщо `foreach` послідовно витягує елементи контейнера і тільки для читання, то в циклі `for` ми можемо перескакувати на кілька елементів вперед залежно від збільшення лічильника, а також можемо змінювати елементи:

```csharp
int[] numbers = { 1, 2, 3, 4, 5 };
for (int i = 0; i < numbers.Length; i++)
{
    numbers[i] = numbers[i] * 2;
    Console.WriteLine(numbers[i]);
}
```

Також можна використовувати й інші види циклів, наприклад `while`:

```csharp
int[] numbers = { 1, 2, 3, 4, 5 };
int i = 0;
while (i < numbers.Length)
{
    Console.WriteLine(numbers[i]);
    i++;
}
```

## Багатовимірні масиви

Масиви характеризуються таким поняттям як ранг чи кількість вимірів. Вище ми розглядали масиви, які мають один вимір (тобто їх ранг дорівнює 1) - такі масиви можна представляти у вигляді ряду (рядки чи стовпця) елемента. Але масиви також бувають багатовимірними. У таких масивів кількість вимірювань (тобто ранг) більша за 1.

Масиви які мають два виміри (ранг дорівнює 2) називають двовимірними. Наприклад, створимо одномірний та двомірний масиви, які мають однакові елементи:

```csharp
int[] nums1 = new int[] { 0, 1, 2, 3, 4, 5 };

int[,] nums2 = { { 0, 1, 2 }, { 3, 4, 5 } };
```

Візуально обидва масиви можна представити так:

Одновимірний масив `nums1`

![Одновимірний масив nums1 з оригінального документа](_assets/_docx/image25.png)

Двовимірний масив `nums2`

![Двовимірний масив nums2 з оригінального документа](_assets/_docx/image26.png)

Оскільки масив `nums2` двовимірний, він є простою таблицею. Усі можливі способи визначення двовимірних масивів:

```csharp
int[,] nums1;
int[,] nums2 = new int[2, 3];
int[,] nums3 = new int[2, 3] { { 0, 1, 2 }, { 3, 4, 5 } };
int[,] nums4 = new int[,] { { 0, 1, 2 }, { 3, 4, 5 } };
int[,] nums5 = new[,] { { 0, 1, 2 }, { 3, 4, 5 } };
int[,] nums6 = { { 0, 1, 2 }, { 3, 4, 5 } };
```

Масиви можуть мати і більшу кількість вимірів. Оголошення тривимірного масиву могло б виглядати так:

```csharp
int[,,] nums3 = new int[2, 3, 4];
```

Відповідно можуть бути і чотиривимірні масиви та масиви з великою кількістю вимірювань. Але на практиці зазвичай використовуються одновимірні та двомірні масиви.

Певну складність може становити перебір багатовимірного масиву. Насамперед треба враховувати, що довжина такого масиву - це сукупна кількість елементів.

```csharp
int[,] numbers = { { 1, 2, 3 }, { 4, 5, 6 } };
foreach (int i in numbers)
    Console.Write($"{i} ");
```

У даному випадку довжина масиву `numbers` дорівнює 6. І цикл `foreach` виводить всі елементи масиву в рядок:

![Рисунок з оригінального документа](_assets/_docx/image27.png)

Але якщо ми хочемо окремо пробігтися по кожному рядку в таблиці? В цьому випадку треба одержати кількість елементів у розмірності. Зокрема, кожен масив має метод `GetUpperBound(номер_розмірності)`, який повертає індекс останнього елемента у певній розмірності. І якщо ми говоримо безпосередньо про двомірний масив, то перша розмірність (з індексом 0) по суті це і є таблиця. І за допомогою виразу

```csharp
numbers.GetUpperBound(0) + 1
```

можна отримати кількість рядків таблиці, поданої двовимірним масивом. А через

```csharp
numbers.Length / кількість_рядків
```

можна отримати кількість елементів у кожному рядку:

```csharp
int[,] numbers = { { 1, 2, 3 }, { 4, 5, 6 } };

int rows = numbers.GetUpperBound(0) + 1; // кількість рядків
int columns = numbers.Length / rows; // кількість стовпців
// або так
// int columns = numbers.GetUpperBound(1) + 1;

for (int i = 0; i < rows; i++)
{
    for (int j = 0; j < columns; j++)
    {
        Console.Write($"{numbers[i, j]} \t");
    }
    Console.WriteLine();
}
```

## Масив масивів

Від багатовимірних масивів треба відрізняти масив масивів або так званий "зубчастий масив":

```csharp
int[][] nums = new int[3][];
nums[0] = new int[2] { 1, 2 };
// Виділяємо пам'ять для першого підмасиву
nums[1] = new int[3] { 1, 2, 3 };
// Виділяємо пам'ять для другого підмасиву
nums[2] = new int[5] { 1, 2, 3, 4, 5 };
// виділяємо пам'ять для третього підмасиву
```

Тут дві групи квадратних дужок вказують, що це масив масивів, тобто такий масив, який у свою чергу містить інші масиви. Причому довжина масиву вказується лише у перших квадратних дужках, всі наступні квадратні дужки мають бути порожні: `new int[3][]`. В даному випадку у нас масив `nums` містить три масиви. Причому розмірність кожного з цих масивів може збігатися.

Альтернативне визначення масиву масивів:

```csharp
int[][] numbers =
{
    new int[] { 1, 2 },
    new int[] { 1, 2, 3 },
    new int[] { 1, 2, 3, 4, 5 }
};
```

Зубчастий масив `nums`

![Зубчастий масив nums з оригінального документа](_assets/_docx/image29.png)

Використовуючи вкладені цикли, можна перебирати зубчасті масиви. Наприклад:

```csharp
int[][] numbers = new int[3][];
numbers[0] = new int[] { 1, 2 };
numbers[1] = new int[] { 1, 2, 3 };
numbers[2] = new int[] { 1, 2, 3, 4, 5 };
foreach (int[] row in numbers)
{
    foreach (int number in row)
    {
        Console.Write($"{number} \t");
    }
    Console.WriteLine();
}

// Перебір за допомогою циклу for
for (int i = 0; i < numbers.Length; i++)
{
    for (int j = 0; j < numbers[i].Length; j++)
    {
        Console.Write($"{numbers[i][j]} \t");
    }
    Console.WriteLine();
}
```

Основні поняття масивів

Підсумовуємо основні поняття масивів:

- Ранг (rank): кількість вимірювань масиву
- Довжина виміру (dimension length): довжина окремого виміру масиву
- Довжина масиву (array length): кількість всіх елементів масиву

Наприклад, візьмемо масив:

```csharp
int[,] numbers = new int[3, 4];
```

Масив `numbers` двомірний, тобто він має два виміри, тому його ранг дорівнює 2. Довжина першого виміру - 3, довжина другого виміру - 4. Довжина масиву (тобто загальна кількість елементів) - 12.

# 2.15. Завдання з масивами

Познайомившись із циклами, змінними, умовними конструкціями та масивами, розглянемо кілька завдань для роботи з масивами.

## Кількість позитивних чисел

Знайдемо кількість позитивних чисел у масиві:

```csharp
int[] numbers = { -4, -3, -2, -1, 0, 1, 2, 3, 4 };
int result = 0;
foreach (int number in numbers)
{
    if (number > 0)
    {
        result++;
    }
}
Console.WriteLine($"Кількість елементів більше нуля: {result}");
```

Тут створюємо допоміжну змінну `result`, яка міститиме кількість позитивних чисел. У циклі переходимо масивом і, якщо його елемент більше нуля, додаємо до змінної `result` одиницю.

## Інверсія масиву

Друге завдання - інверсія масиву, тобто переворот його у зворотному порядку:

```csharp
int[] numbers = { -4, -3, -2, -1, 0, 1, 2, 3, 4 };

int n = numbers.Length; // Довжина масиву
int k = n / 2; // середина масиву
int temp; // Допоміжний елемент для обміну значеннями
for (int i = 0; i < k; i++)
{
    temp = numbers[i];
    numbers[i] = numbers[n - i - 1];
    numbers[n - i - 1] = temp;
}
foreach (int i in numbers)
{
    Console.Write($"{i}\t");
}
```

Оскільки нам треба змінювати елементи масиву, для цього використовується цикл `for`. Алгоритм розв'язання задачі має на увазі перебір елементів до середини масиву, яка в програмі представлена змінною `k`, та обмін значень елемента, який має індекс `i`, та елемента з індексом `n - i - 1`.

## Програма сортування масиву

Тепер візьмемо завдання складніше - найпростіше сортування масиву:

```csharp
int[] nums = { 54, 7, -41, 2, 4, 2, 89, 33, -5, 12 };

// сортування
int temp;
for (int i = 0; i < nums.Length - 1; i++)
{
    for (int j = i + 1; j < nums.Length; j++)
    {
        if (nums[i] > nums[j])
        {
            temp = nums[i];
            nums[i] = nums[j];
            nums[j] = temp;
        }
    }
}

// Виведення
Console.WriteLine("Виведення відсортованого масиву");
for (int i = 0; i < nums.Length; i++)
{
    Console.WriteLine(nums[i]);
}
```

Для сортування масиву виконуємо проходи масивом і порівнюємо елементи. Оскільки нам треба послідовно порівнювати кожен елемент масиву з кожним (за винятком порівняння із самим собою), то тут застосовуються вкладений цикл.

У зовнішньому циклі ми беремо елемент, який порівнюватимемо:

```csharp
for (int i = 0; i < nums.Length - 1; i++)
```

Далі запускаємо вкладений цикл, який починається, з наступного елемента, і з якого вилучаємо елементи, з якими порівнюватимемо той елемент, який береться з масиву в зовнішньому циклі:

```csharp
for (int j = i + 1; j < nums.Length; j++)
```

Якщо елемент з меншим індексом більший за елемент з великим індексом, то міняємо елементи місцями.

```csharp
if (nums[i] > nums[j])
{
    temp = nums[i];
    nums[i] = nums[j];
    nums[j] = temp;
}
```

Наприкінці виводимо всі елементи.
