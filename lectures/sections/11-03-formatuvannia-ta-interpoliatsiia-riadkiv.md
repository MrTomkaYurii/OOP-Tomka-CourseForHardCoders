## 11.3. Форматування та інтерполяція рядків

### Форматування рядків

При виведенні рядків у консолі за допомогою методу `Console.WriteLine` для вбудовування значень у рядок ми можемо застосовувати форматування замість конкатенації:

```csharp
string name = "Tom";
int age = 23;

Console.WriteLine("Ім'я: {0} Вік: {1}", name, age);
// консольний вивід
// Ім'я: Tom Вік: 23
```

У рядку `"Ім'я: {0} Вік: {1}"` на місце `{0}` і `{1}` потім будуть вставлятися в порядку проходження змінних `name` і `age`.

Те саме форматування в рядку ми можемо зробити не тільки в методі `Console.WriteLine`, але і в будь-якому місці програми за допомогою методу `string.Format`:

```csharp
string name = "Tom";
int age = 23;
string output = string.Format("Ім'я: {0} Вік: {1}", name, age);
Console.WriteLine(output);
```

Метод `Format` приймає рядок із плейсхолдерами типу `{0}`, `{1}` і т.д., а також набір аргументів, що вставляються на місце даних плейсхолдерів. У результаті генерується новий рядок.

### Специфікатори форматування

У методі `Format` можуть використовуватися різні специфікатори та описувачі, які дозволяють налаштувати виведення даних. Розглянемо основні описники. Усі формати:

### Форматування валюти

Для форматування валюти використовується описник `"C"`:

```csharp
double number = 23.7;

string result = string.Format("{0:C0}", number);
Console.WriteLine(result); // 24 р.

string result2 = string.Format("{0:C2}", number);
Console.WriteLine(result2); // 23,70 р.
```

Число після описувача вказує, скільки чисел буде використовуватись після роздільника між цілою та дробовою частиною. Під час виведення також додається позначка грошового знака для поточної культури комп'ютера. Залежно від локалізації поточної операційної системи, результат може відрізнятися. Також зверніть увагу на округлення у першому прикладі.

### Форматування цілих чисел

Для форматування цілих значень застосовується описник `"d"`:

```csharp
int number = 23;

string result = string.Format("{0:d}", number);
Console.WriteLine(result); // 23

string result2 = string.Format("{0:d4}", number);
Console.WriteLine(result2); // 0023
```

Число після описувача вказує, скільки цифр буде у числовому значенні. Якщо у вихідному числі цифр менше, до нього додаються нулі.

### Форматування дробових чисел

Для форматування дробових чисел використовується описник `F`, число після якого вказує, скільки знаків буде використовуватися після роздільника між цілою та дробовою частиною. Якщо вихідне число - ціле, то до нього додаються роздільник та нулі.

```csharp
int number = 23;
string result = string.Format("{0:f}", number);
Console.WriteLine(result); // 23,00

double number2 = 45.08;
string result2 = string.Format("{0:f4}", number2);
Console.WriteLine(result2); // 45,0800

double number3 = 25.07;
string result3 = string.Format("{0:f1}", number3);
Console.WriteLine(result3); // 25,1
```

### Формат відсотків

Описувач `"P"` визначає відображення відсотків. Чисельний специфікатор, що використовується з ним, вказує, скільки знаків буде після коми:

```csharp
decimal number = 0.15345m;
Console.WriteLine("{0:P1}", number); // 15,3%
```

### Настроювані формати

Використовуючи знак `#`, можна настроїти формат виводу. Наприклад, нам потрібно вивести деяке число у форматі телефону `+х (ххх)ххх-хх-хх`:

```csharp
long number = 19876543210;
string result = string.Format("{0:+# (###) ###-##-##}", number);
Console.WriteLine(result); // +1 (987) 654-32-10
```

### Метод ToString

Метод `ToString()` не тільки отримує рядковий опис об'єкта, а й може здійснювати форматування. Він підтримує ті ж описувачі, що використовуються у методі `Format`:

```csharp
long number = 19876543210;
Console.WriteLine(number.ToString("+# (###) ###-##-##")); // +1 (987) 654-32-10

double money = 24.8;
Console.WriteLine(money.ToString("C2")); // 24,80 р.
```

### Інтерполяція рядків

Інтерполяція рядків має спростити форматування рядків. Так, перепишемо приклад із виведенням значень змінних у рядку:

```csharp
string name = "Tom";
int age = 23;

Console.WriteLine($"Ім'я: {name} Вік: {age}");
// консольний вивід
// Ім'я: Tom Вік: 23
```

Знак долара перед рядком вказує, що здійснюватиметься інтерполяція рядків. Усередині рядка знову ж таки використовуються плейсхолдери `{...}`, тільки всередині фігурних дужок вже можна безпосередньо писати ті висловлювання, які ми хочемо вивести.

Інтерполяція насправді представляє більш лаконічне форматування. При цьому всередині фігурних дужок ми можемо вказувати як властивості, а й різні висловлювання мови C#:

```csharp
int x = 8;
int y = 7;
string result = $"{x} + {y} = {x + y}";
Console.WriteLine(result); // 8 + 7 = 15
```

Також усередині фігурних дужок можна виконувати складніші вирази, наприклад, викликати методи:

```csharp
int x = 8;
int y = 7;
string result = $"{x} * {y} = {Multiply(x, y)}";
Console.WriteLine(result); // 8 * 7 = 56

int Multiply(int a, int b) => a * b;
```

Уже всередині рядка можна використовувати форматування. У цьому випадку ми можемо застосовувати ті самі описувачі, що і в методі `Format`. Наприклад, виведемо номер телефону у форматі `+x xxx-xxx-xx-xx`:

```csharp
long number = 19876543210;
Console.WriteLine($"{number:+# ### ### ## ##}"); // +1 987 654 32 10
```

Додаємо простір до та після форматованого виводу:

```csharp
string name = "Tom";
int age = 23;

Console.WriteLine($"Ім'я: {name,-5} Вік: {age}"); // пробіли після
Console.WriteLine($"Ім'я: {name,5} Вік: {age}");  // пробіли до
```
