## 1.2. .NET Framework та .NET 6

Варто зазначити, що .NET тривалий час розвивався переважно як платформа для Windows під назвою .NET Framework. У 2019 році вийшла остання версія цієї платформи - .NET Framework 4.8. Вона більше не розвивається.

З 2014 року Microsoft став розвивати альтернативну платформу - .NET Core, яка вже призначалася для різних платформ і повинна була увібрати в себе всі можливості застарілого .NET Framework і додати нову функціональність. Потім Microsoft послідовно випустив ряд версій цієї платформи: .NET Core 1, .NET Core 2, .NET Core 3, .NET 5. І поточною версією є платформа .NET 6, що розглядається в цьому посібнику. На відміну від .NET Framework, яка була переважно платформою для Windows, .NET 6 є кросплатформовою. У цьому посібнику йтиметься про C# 10 у зв'язку з .NET 6, оскільки це актуальна платформа.
