## 9.5. Реляційний та логічний патерни

У C# 9.0 до мови були додані додаткові патерни - реляційний (relational pattern) і логічний (logical pattern) патерни.

Реляційний патерн дозволяє порівняти значення, що передається в конструкцію, з деякими значеннями за допомогою операцій порівняння. Наприклад, залежно від суми вкладу, відсотки за цим вкладом можуть відрізнятися. Розрахуємо суму відсотків залежно від суми вкладу з використанням реляційного патерну:

```csharp
decimal Calculate(decimal sum)
{
    return sum switch
    {
        <= 0 => 0,              // якщо sum менше або дорівнює 0, повертаємо 0
        < 50000 => sum * 0.05m, // якщо sum менше 50000, повертаємо sum * 0.05m
        < 100000 => sum * 0.1m, // якщо sum менше 100 000, повертаємо sum * 0.1m
        _ => sum * 0.2m         // в інших випадках повертаємо sum * 0.2m
    };
}
```

Застосування:

```csharp
Console.WriteLine(Calculate(-200));     // 0
Console.WriteLine(Calculate(0));        // 0
Console.WriteLine(Calculate(10000));    // 500
Console.WriteLine(Calculate(60000));    // 6000
Console.WriteLine(Calculate(200000));   // 40000
```

Логічний патерн дозволяє використовувати логічні оператори and (логічне множення чи операція логічного І) та or (логічне складання чи операція логічного АБО) для об'єднання операцій порівняння. Наприклад, передамо в метод вік і повернемо відповідне повідомлення:

```csharp
string CheckAge(int age)
{
    return age switch
    {
        < 1 or > 110 => "Недійсний вік",       // якщо age більше 110 або менше 1
        >= 1 and < 18 => "Доступ заборонено", // якщо age дорівнює чи більше 1 і менше 18
        _ => "Доступ дозволено"               // в інших випадках
    };
}
```

Застосування:

```csharp
Console.WriteLine(CheckAge(200)); // Недійсний вік
Console.WriteLine(CheckAge(0));   // Недійсний вік
Console.WriteLine(CheckAge(17));  // Доступ заборонено
Console.WriteLine(CheckAge(18));  // Доступ дозволено
```

Ще один логічний оператор - not використовується для заперечення і повертає true, якщо його аргумент дорівнює false:

```csharp
string CheckAge(int age) => age switch
{
    not 33 => "Звичайний вік", // якщо age не дорівнює 33
    _ => "Вам 33 роки"         // в інших випадках, тобто якщо age = 33
};
```
